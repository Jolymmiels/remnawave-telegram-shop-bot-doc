## Why Remnawave Telegram Shop?

The bot provides a complete solution for selling VPN subscriptions through Telegram:

- **Multiple Payment Methods**: YooKassa, CryptoPay, Telegram Stars, and Tribute
- **Automatic Notifications**: Remind users 3 days before subscription expires
- **Squad Management**: Flexible configuration of server access
- **Trial System**: Offer trial subscriptions to attract new users
- **Multi-language Support**: Full support for Russian and English
- **Security Features**: User filtering, whitelist/blacklist support
